## 学習メモ
※学習した内容をここに書き留めておきましょう

### コンポーネントとは
- UIを構成する部品のこと
- 関数として定義し、JSXを返す（関数名は大文字始まり）
- `<View>`（箱）や `<Text>`（文字）などのタグを組み合わせて作る
- `export default` で他のファイルから使えるようにする
- コンポーネントは入れ子にして組み合わせられる
- `style={{ color: 'blue', fontSize: 24 }}` で見た目を変えられる

### Propsとは
- コンポーネントに渡す「引数」のこと
- 外から値を渡すことでコンポーネントを使い回せる
- `<Greeting name="アキラ" />` のように属性として渡す
- 受け取る側は `({ name })` のように分割代入で受け取る
- TypeScriptでは型指定が必要（例：`{ name: string, age: number }`）
- 数値の型は `int` ではなく `number`

### Stateとは
- コンポーネント自身が持つ「変化するデータ」のこと
- `useState` フックで定義する
- `const [値, 更新関数] = useState(初期値)` の形で書く
- 更新関数を呼ぶと画面が自動で再描画される
- Propsは外から渡される値、Stateは自分自身が持つ値

### Props と State を組み合わせる
- Propsで外から渡す値、Stateで自分が管理する値を組み合わせて動きのある部品が作れる
- 同じコンポーネントを複数使ってもStateはそれぞれ独立している
- TypeScriptでPropsの型は `type Props = { title: string; }` のように定義できる

### レイアウトを整える（StyleSheet）
- `StyleSheet.create({})` でスタイルをまとめて定義する
- `style={styles.xxx}` でコンポーネントに適用する
- `flexDirection: 'row'` で要素を横並びにする
- `justifyContent: 'space-around'` で均等配置
- `textAlign: 'center'` で文字を中央揃え
- `borderWidth` / `borderColor` / `borderRadius` で枠線を作る
