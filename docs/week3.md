### AsyncStorage とは
- 入力されたデータをアプリのストレージに保存することができる
- アプリを閉じても消えないデータの箱
- 保存する await AsyncStorage.setItem('キー', ''); キーは保存場所の名前、値は保存したいデータ
- 読み込む const value = await AsyncStorage.getItem('キー'); 
- 配列・オブジェクトを扱うとき AsyncStorage は文字列しか保存できないので変換が必要  
　// 保存するとき JSON.stringify([1, 2, 3])  // → '[ 1, 2, 3 ]'（文字列に変換）
　// 読み込むとき JSON.parse('[1, 2, 3]')    // → [1, 2, 3]（元に戻す）
- useEffect と組み合わせる 画面が起動したときに自動で読み込むには useEffect を使う　
useEffect(() => {                       
      const load = async () => {                                               
        const value = await AsyncStorage.getItem('キー');
            if (value !== null) {
                setData(JSON.parse(value) ?? []);
            }                                                                                    
        };
      load();   
}, []);
