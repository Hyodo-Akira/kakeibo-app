※学習した内容をここに書き留めておきましょう

### TextInput とは
- ユーザーが文字を入力できるコンポーネント（Web の `<input>` 相当）
- `value` と `onChangeText` をセットで使う
- `keyboardType="numeric"` で数字専用キーボードにできる
- 同じ State を `value` にも表示にも使えばリアルタイム反映できる
- `{条件 && <Component />}` で条件付きレンダリングができる（JSX内では if が使えないため）
- `onChangeText` の引数には入力後の文字列全体が渡される

### FlatList とは
- PHPでいう<li>や<ul>のようなものでリストを作るときに用いる
- 関数でリストに入る値を定義し、FlatListの中にdataやrenderItem,keyExtractorを用いて記述する
- renderItem の { item } に配列の各要素が1つずつ入ってくる
- keyExtractor は各要素を一意に識別するためのキーを返す（React の key 相当）
